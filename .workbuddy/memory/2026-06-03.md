## Expo Go 卡死修复 — TransitionScreen three.js 兼容问题

- 现象：Expo Go 打开 app 后卡死在"消息"页，底部报错 `Scripts "build/three.js" and "build/thre..."`，所有页面跳转失效
- 根因：`client/screens/TransitionScreen.js` 引入 `three` 和 `expo-gl`，这两个库不兼容 Expo Go，Metro bundler 报错导致整个 JS bundle 崩溃
- 修复：
  1. 用纯 React Native `Animated` + 爱心参数方程重写 `TransitionScreen.js`，视觉效果保持（36个彩色粒子汇聚爱心→呼吸→散开）
  2. 从 `client/package.json` 移除 `three` 和 `expo-gl` 依赖
  3. 需在 `client/` 目录执行 `npx expo start --clear` 清除缓存重启
